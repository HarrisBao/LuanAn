# HyperRAG Thesis Demo

Dự án demo hệ thống RAG/HyperRAG phục vụ luận văn thạc sĩ.

## Chức năng chính

- Nạp dữ liệu tài liệu học tập
- Tạo vector index cục bộ
- Truy xuất tài liệu liên quan theo câu hỏi
- Sinh câu trả lời có bằng chứng
- Giao diện demo bằng Streamlit

python -m venv .venv
